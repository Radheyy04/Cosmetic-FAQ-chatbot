# Cosmetic-FAQ-chatbot
An AI-based FAQ chatbot developed using Python, Flask, NLP, and Machine Learning.   The chatbot answers customer queries related to cosmetic and beauty products using TF-IDF Vectorization and Cosine Similarity.

## Features

- NLP text preprocessing
- FAQ matching
- AI chatbot responses
- Flask backend
- Interactive UI

## Technologies Used

- Python
- Flask
- NLTK
- Scikit-learn
- HTML/CSS/JavaScript

## Installation

```bash
pip install flask nltk scikit-learn
```

## Run Project

```bash
python app.py
```
